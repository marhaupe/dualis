Simple hack to check your grades.

# Installation

Download the [archive](https://github.com/marhaupe/dualis/releases) and extract it somewhere your `$PATH` is set to, for example to `/usr/local/bin`:

```bash
tar -C /usr/local/bin -xzf <archive>
```
